# TG Tools — Инструкция для Windows

## ⚡ Быстрый старт (рекомендуется)

1. Распакуй архив в любую папку
2. Дважды кликни **`install_and_run.bat`**
3. При первом запуске автоматически установятся все нужные пакеты
4. Приложение откроется само

> При первом запуске Windows может показать предупреждение.
> Нажми "Подробнее" → "Выполнить в любом случае"

---

## 🔨 Собрать .EXE (чтобы запускать без Python)

1. Сначала запусти `install_and_run.bat` хотя бы один раз
2. Затем запусти **`build.bat`**
3. Через 2-3 минуты в папке `dist\` появится `TGTools.exe`
4. Этот `.exe` можно перекинуть куда угодно

---

## 📋 Требования

- Windows 10 / 11
- Python 3.11+ → [скачать здесь](https://python.org/downloads)
  - ⚠️ При установке Python поставь галочку **"Add Python to PATH"**

---

## 🔐 Авторизация в приложении

1. Зайди на [my.telegram.org](https://my.telegram.org)
2. Войди своим номером телефона
3. Нажми "API development tools"
4. Скопируй `App api_id` и `App api_hash`
5. Вставь в приложение → Авторизация

---

## 📁 Структура папок

```
TGTools/
├── app.py                ← основной скрипт
├── install_and_run.bat   ← запуск (используй это)
├── build.bat             ← сборка в .exe
├── README_WINDOWS.txt    ← этот файл
├── session.session       ← создаётся автоматически (сессия Telegram)
├── logs/                 ← логи работы
├── downloads/            ← скачанные медиа
└── exports/              ← экспорт данных
```
