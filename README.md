# TG Tools — Telegram Automation Suite

## Установка (один раз)

```bash
pip install telethon customtkinter pillow qrcode
```

## Запуск

```bash
python app.py
```

## Разделы приложения

### 🔐 Авторизация
- Введи API_ID и API_HASH с [my.telegram.org](https://my.telegram.org)
- Нажми «Войти по QR»
- Открой Telegram → Настройки → Устройства → Подключить → отсканируй

### 🔗 Рефералы
- Вставь ссылки (по одной на строку)
- Нажми «Запустить»
- Скрипт автоматически:
  - Подписывается на каналы
  - Запускает ботов
  - Подписывается на спонсоров из кнопок
  - Нажимает «✅ Я подписан»
  - Выходит из каналов-спонсоров
  - Блокирует ботов

### 🧹 Очистка
- Выходит из всех каналов/групп
- Блокирует всех ботов
- Укажи исключения (ID через запятую)

### 🛠 Инструменты
- Скачать медиа из чата
- Экспорт истории в JSON
- Статистика диалогов

### ⚙️ Настройки
- Таймауты и паузы

## Структура папок

```
tg_app/
├── app.py
├── requirements.txt
├── session.session      ← создаётся автоматически
├── logs/                ← логи работы
├── downloads/           ← скачанные медиа
└── exports/             ← экспортированные данные
```
